# fastAPI-example

## Introduction

This Project Is Practice FastAPI Framework


#### Getting Started

- Enter Project Directory
```
poetry install
```

- poetry run uvicorn {File Name}:{You Declare The FastAPI} --reload
```
poetry run uvicorn app:app --reload
```
